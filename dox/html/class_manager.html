int Manager::ReadParametersFile ( int  modelNumber)

Reads chosen parameters of ARX object from file.

Obtains chosen three groups of parameters separated with semicolon (A;B;nk) User specifies number of desired model, each model is placed in a new line. Params sets are separated by semicolon, and each parameter is separated by whitespace.

Parameters
modelNumberNumber of chosen model from file
Returns
1 if everything worked correctly, 0 if some error occured
